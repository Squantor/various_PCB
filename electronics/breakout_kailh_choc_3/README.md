# 4 button kailh choc breakout
Simple test breakout for the kailh choc buttons
# HSI (Hardware Software Interface)
6 pin connector:
1. LED cathodes row 1
2. LED cathodes row 2
3. Switch Row 1
4. Switch Row 2
5. Column 2
6. Column 1
